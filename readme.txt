Take Control over text copy action in your blog. 

== Description ==

With Control Copy you can control, limit and manipulate the copied text or text copy attempts made by visitors. 

**New on version 1.3 -** Limit WPCC to Specific Posts/Pages!

**On the current version, you can :**
 
* Limit text copy to Logged In users only.
* Disable text copy entirely.
* add custom text (or html) at the end of the copied text (for posts or site-wide).
* custom text can include - urls, links, post author, year, title, etc.
* allow members get a clean text copy.
* limit WPCC to posts only.
* display elegant alerts. 
* you can limit WPCC to specific posts/pages (ids);

All done easily from the backend.

If you are missing a feature – drop us a line, we’d love to extend this plugin for your needs!

Control Copy is hooked to wp_footer(), so make sure your theme is calling wp_footer.

You can find detailed information with examples in [Control Copy](http://omniwp.com/plugins/wpcc-a-wordpress-plugin/ "Control Copy's Page").

== Installation ==

1. Upload the plugin zip content to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Thats it!
